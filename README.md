# Skill Vault

一个好用的Skill 合集。

Skill 是写给 AI Agent 的结构化指令文件，告诉 Agent 在特定场景下该做什么、怎么做。当用户的提问匹配到 Skill 的描述时，Agent 会自动加载对应的 Skill 并按其指引完成任务。

---

## 当前 Skill 列表

| Skill | 功能 |
|-------|------|
| [code-weave](./code-weave/) | 梳理代码逻辑，生成中文标注的 Mermaid 流程图 + 节点功能文档 |
| [java-coder](./java-coder/) | 生成符合阿里 Java 开发规范的 Spring Boot + MyBatis 代码 |
| [opencode-docs](./opencode-docs/) | OpenCode 文档问答助手，回答安装、配置、模型、MCP 等相关问题 |
| [skill-creator](./skill-creator/) | 创建新 Skill、迭代优化、跑评测、打包分发的元 Skill |
